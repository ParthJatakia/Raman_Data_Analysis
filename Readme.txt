Please read before using.

Find_Ratio_sp3sp2.py

1.Data should be in the form of txt and the first column should be the energies at which     the measurement is done. The txt file should be in the same folder as the python script
2.The columns should not contain anything else except the data. Columns should not contain   the position of the measurement. The position will be asked during the runtime.
3.The graph will be stored as an image named ratio_of_sp3sp2.png in the same folder.
